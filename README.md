## üöÄ Key Features

- **Animated SVG Turbulence**: Leverages `feTurbulence` and `feDisplacementMap` for fluid, organic electric pulse animations.
- **Glassmorphism Drawer**: A sleek, semi-transparent customization panel with real-time blur and depth effects.
- **Dynamic CSS variables**: Seamlessly updates styles and aura colors without reloading, providing a smooth user experience.
- **Interactive Customizer**: Toggleable side drawer allows you to tweak:
    - **Base Frequency**: Adjust the "jitter" and speed of the electricity.
    - **Displacement Scale**: Control the intensity and spread of the electric arcs.
    - **Aura Color**: Real-time color picker for the card's glow.
    - **Glow Intensity**: Fine-tune the spread of the luminous layers.
    - **Card Rounding**: Smoothly adjust corner radii.
- **Fully Responsive**: Intelligent layout CSS and media queries ensure the card looks stunning on mobile and desktop.

## üåì Design Philosophy

This project is an exploration of **Interactive Motion Aesthetics**. By combining **mathematical noise** (SVG turbulence) with **modern UI surfaces** (Glassmorphism), we achieve a visual effect that feels both synthetic and organic. The interface is designed to respond to user input instantly, turning a static card into a controllable digital kinetic sculpture.
